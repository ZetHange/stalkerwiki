# Заброшенные проекты
Описание проектов, которые по каким-либо причинам были заброшены
## Project Stalker
Проект по переносу ТЧ на Android, основан примерно в 2015 году. Project STALKER - это целый мир, который группа разработчиков перенесла легендарной игры на андроид.
### Краткое описание:
Присутствует почти вся основная сюжетная линия локаций "Кордон" и "Арена" несколько побочных квестов и пасхалок.
Билд 1.7.0 является первой полноценной игровой сборкой проекта с сюжетной линией, а не тестом каких-то отдельно взятых систем.
