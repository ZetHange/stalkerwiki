# FAQ: S.T.A.L.K.E.R. Mobile Edition
*Стрындено от сюда:* [Ссылка](https://vk.com/topic-184902497_44410327)
В: Системные требования?
О: Минимальные - Android 4.4 и больше, 1gb RAM, 4 x 1.3GHz

В: Дата выхода?
О: Дата будет опубликована после 25.05.2020

В: Есть ли билды? Как поиграть?
Будет ЗБТ. Регистрация начнется 25.05.2020

В: Как попасть в ЗБТ?
О: 25 мая начнется регистрация. По окончании часть участников пройдет в ЗБТ. Четкие критерии будут 25 мая.

В: Сколько будет весить?
О: Около 1.5gb

В: Сколько локаций?
О: Все. Будет полная копия оригинала.

В: Будет ли мультиплеер?
О: Да. Классический. Как в оригинале.

В: Будут ли пасхалки?
О: Нет. Будет полная копия оригинала.

В: Будут все концовки?
О: Да. Будет полная копия оригинала.

В: Будет ли исправление багов оригинальной части.
О: Основные баги будут исправлены.

В: Будет ли ЧН? ЗП?
О: Да, будут все части трилогии.

В: Будет ли кооператив?
О: Нет.

В: Есть ли поддержка геймпада?
О: Да.

В: Будут ли моды?
О: Да. Будет Mod Development Kit и документация к нему.
Будет возможность создания модов и адаптации уже имеющихся.

В: Будет ли проект платным?
О: Проект бесплатный.

В: Где кнопка инвентаря, фонарика... где все?
О: Инвентарь открывается через открытие селектора выбора оружия.
Там же и остальные кнопки.

В: Будет ли кастомизация управления? Можно ли добавить вторую кнопку стрельбы?
О: Да. Все индивидуально подгоняется в настройках.

В: Как почомь проекту?
О: Если вы матерый разработчик и подходите под критерии - можете помочь с кодом. Так же есть реквизиты для финансовой поддержки проекта. И самое простое - лайк, репост, комментарий.
